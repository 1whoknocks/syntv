# SynoCam APK hosting branch

Prebuilt APKs for sideloading SynoCam onto Fire TV via the Downloader app.

| File | ABI | Use for |
|------|-----|---------|
| `synocam-arm64-v8a-v0.2.0.apk`   | arm64-v8a   | Fire TV Stick 4K / 4K Max / Cube, Stick 3rd gen (64-bit) |
| `synocam-armeabi-v7a-v0.2.0.apk` | armeabi-v7a | **Any** Fire TV (32-bit *and* 64-bit) — universal fallback |

Direct links:
- https://raw.githubusercontent.com/1whoknocks/syntv/downloads/synocam-arm64-v8a-v0.2.0.apk
- https://raw.githubusercontent.com/1whoknocks/syntv/downloads/synocam-armeabi-v7a-v0.2.0.apk

App source + README live on `main`.
